To run the Task :

1.Download or clone the github repo , extract it and move in to the directory in the terminal.

2.run "npm install"

3.run "node server.js"

4.Open POSTMAN app in google chrome .

5.Test Links :
  for users :
  1. http://localhost:8080/users/create
  2. http://localhost:8080/users/readall
  3. http://localhost:8080/users/readsingle/:userID (Copy paste _id in userID)
  4. http://localhost:8080/users/update/:userID
  5. http://localhost:8080/users/delete/:userID

  for Pictures :
  1. http://localhost:8080/pic/create
  2. http://localhost:8080/pic/readall
  3. http://localhost:8080/pic/readsingle/:picID (copy paste _id in picID)
  4. http://localhost:8080/pic/update/:picID
  5. http://localhost:8080/pic/delete/:picID
